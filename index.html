Documentação

SOBRE

Este documento apresenta uma aplicação web que facilita o levantamento de requisitos de novos sistemas. Ela auxilia o levantador de requisitos, permitindo a criação de protótipos com base nas informações inseridas pelo usuário. A aplicação utiliza a API do ChatGPT para gerar o código HTML correspondente ao protótipo, acelerando a visualização e validação dos requisitos, promovendo uma colaboração mais efetiva entre as partes interessadas. O documento fornece uma visão geral da aplicação, suas funções e considerações técnicas. Além disso, inclui instruções detalhadas para a instalação, configuração e uso da aplicação.
